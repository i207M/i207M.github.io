血染钟楼-游戏经验

发表于 2022-05-04 分类于 Game

五一假期玩的很带劲，遂写一下自己的心得：

  • 开局F4信息位直接给出信息，但是有的信息需要先让别人说，这样可能开局揪出坏人。有可能被开局盲毒。
  • 士兵应该跳僧侣抗刀。
  • 恶魔可以跳共情者，尤其是酒鬼共情者。
  • 爪牙可以跳杀手，指向恶魔。
  • 坏人可以跳：
    • 没啥用的角色（容易后期“留着也没用”）：开局F4，市长，士兵。
    • 守鸦人，这是个好角色，首先它开局不用亮身份，其次它亮了身份之后，谁也不想杀它，这就导致它的身份无法被验证。必要时恶魔自杀，假消息诱导。
    • 信息位，对心理和逻辑挑战很大。
    • 外来者：隐士、管家、酒鬼、圣徒。注意外来者的数量。
  • 坏人不应该太明显地杀掉信息位，这样会验证消息。
  • 守鸦人必要时可以跳僧侣自杀。
  • 僧侣可以跳士兵，但是要赌。
  • 圣女：直接明跳，因为圣女的存在制造了直接出现明好人的可能性，第一天好人多，轮次够，所以处决市民不要有心理负担，反而验出了两个明好人。注意提名顺序。
  • 男爵可以跳信息位，因为它的存在减少了信息位。
  • 间谍也可以跳信息位。
  • 恶魔建议从说书人给出的安全牌中跳，感觉快盘到自己的时候，该自刀就自刀传刀，不要怕死。

Web一把梭

发表于 2022-05-01 分类于 Dev

这里记录与Web开发有关的一些琐碎的知识。

HTTP

Headers

这里记录一些有意思的Header：

  • X-Content-Type-Options 相当于一个提示标志，被服务器用来提示客户端一定要遵循在 Content-Type 首部中对 MIME 类型 的设定，而不能对其进行修改。这就禁用了客户端的 MIME 类型嗅探行为。
  • Content-Disposition 指示回复的内容该以何种形式展示，是以内联的形式（即网页或者页面的一部分），还是以附件的形式下载并保存到本地。

Nginx

与Nginx的奋斗，主要来自于软件工程的部署环节。

Nginx rewrite中的break标志的意思是停止之后的rewrite操作。
